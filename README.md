# Microsoft Caboodle

A kit of essential cross platform API's

## Supported Platforms
 - iOS (10+)
 - Android (4.4+)
 - UWP (Fall Creators Update+)

## Current Features:
 - Permissions
 - Preferences
 - Geolocation
 - Connectivity
 - Device Info
 - SMS / Phone / Email
 - Sharing
 - Text-to-Speech

## Planned Features:
 - Media Picker
 - Browser
